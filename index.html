Project 1: mimik
<-- 001

mimik is a software program for authoring blend-shapes based facial animation. It supports manipulating facial features by dragging sliders, and animating them using splines. This is an old-school approach to doing facial animations for 3D character models, based on FACS (Facial Animation Coding System).

mimik was created using Godot Engine as a framework. Fundamental parts of the program were written using C++, with the rest using GDScript (Godot's own python-like scripting language). For lipsyncing purposes (a.k.a phoneme extraction), it uses the pocketsphinx library.
